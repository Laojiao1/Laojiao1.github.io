基本概念

Add (Element-wise Sum)：可用于 Bottleneck 中的 残差连接（Residual Connection），是残差结构的核心。

残差结构（Residual Structure）是深度学习中的一种重要网络设计理念，主要用于解决深层神经网络训练中的梯度消失和性能退化问题。

通过引入残差块和跳跃连接来解决深度学习训练过程中出现的性能退化问题。

随着网络层数的增加，传统深度网络的训练误差和测试误差可能会上升，ResNet 通过残差学习的方式有效缓解了这一问题。

Element-wise Sum：逐元素相加（注意和concat区别，add 不改变输出通道）

假设有两个矩阵，那么就是两个矩阵对应元素相加

Add 在神经网络里干嘛？

用于残差链接（Residual Connection）

  • 残差连接让网络学习的是差值而不是完整映射，从而缓解梯度消失问题

残差结构：y = F(x) + x

  • x 是输入

  • F(x) 是经过卷积等操作后的输出

  • 最终输出 = 原输入 + 卷积结果

为什么要这样做？—— 解决深层网络训练困难的问题

残差结构允许：梯度直接从后面流回到前面，使得网路更容易学习“恒等映射”。

一句话理解：如果网络学不好，那不如“什么都不变”。

简单来说：激活函数就是对每一个数做一次“变形”

比如卷积输出：[-3, 0.5, 4]

激活输出就是把每个数再加工一下

具体来说：在神经网络中，每一层通常做两件事：

z=Wx+bz=Wx+bz=Wx+b y=ϕ(z)y=ϕ(z)y=ϕ(z)

其中：

  • 第一部分 z 是 线性变换
  • 第二部分 y 是 激活函数

所以本质上：激活函数 = 施加在“线性变换结果”上的非线性算子

激活函数存在的根本意义：打破线性（最核心的一点），如果没有激活函数，单纯一个线性变换，无论做了多少层计算，都还是线性，那网络就会一直退化，无法去拟合复杂的函数。

如果有激活函数，就会让网络具备“函数逼近能力”，经过非线性变换的多层叠加，这种嵌套结构可以逼近任意连续函数，这就是深度网络的表达能力来源。

激活函数一般有三种：

压缩型（Sigmoid、Tanh）

  • 作用：控制数值范围、常用于概率输出
  • 特点：输出有界、梯度可能变小

截断型（ReLU）

  • 作用：过滤负信号、让网络更稀疏
  • 特点：计算简单、负区间梯度为 0

门控型（SiLU）

  • 作用：动态调节信息流、保留部分负信息
  • 特点：平滑、梯度稳定

**从梯度传播的角度理解：**由于深度网络训练依赖反向传播

如果激活函数的导数：

  • 太小 → 梯度消失
  • 太大 → 梯度爆炸
  • 不连续 → 优化不稳定

所以好的激活函数要：

  • 非线性
  • 梯度稳定，不容易导致梯度消失
  • 计算高效

**总结：**激活函数的本质作用，是控制特征如何被“保留、放大或抑制”，它决定了信息在网络中的流动方式。没有激活函数，深度网络将毫无意义

SiLU 和 Sigmoid 的本质与区别

Sigmoid： 只是一个“压缩函数”

SiLU： 是“线性函数 × 压缩门控”

Sigmoid的本质问题：

  • 输出范围固定在 0~1
  • 大部分区域梯度很小
  • 会“压缩”信息

如果你把 Sigmoid 直接当激活函数：

  • 所有输出都会被压缩
  • 网络表达的数值范围受限
  • 深层网络容易梯度变小

它更像一个概率映射器，而不是一个好的特征变换函数。

SiLU 做了一个非常关键的事情：用 Sigmoid 作为“门控系数”去调节 x

  • σ(x) 决定 x 该保留多少
  • x 负责提供幅度

当 x 很大（正数），SiLU保留了线性表达能力

  • σ(x) ≈ 1
  • SiLU ≈ x

当 x 很小（负数），SiLU抑制了无用信号

  • σ(x) ≈ 0
  • SiLU ≈ 0

当 x 在 0 附近，平滑过渡

  • σ(x) ≈ 0.5
  • SiLU ≈ 0.5x

总之：SiLU 是一个带门控的线性单元。

Sigmoid 改变的是“数值范围”，SiLU 改变的是“信息流动方式”

Sigmoid

公式：1/(1+e−x)1/(1 + e^{-x})1/(1+e−x)

  • 它是一个逐元素的非线性压缩型算子，输出范围永远在0~1之间：
    • x 很大 输出接近 1
    • x 很小 输出接近 0
    • x = 0 输出 0.5
  • 导数可以用自身表示
  • forward 结果会在 backward 中被复用

**应用场景：**Sigmod 最常应用于需要输出概率的地方，比如目标检测中检测头的类别概率输出，目标置信度，以及 Pose 可见性

  • 一般应用于二分类问题
  • 多标签分类
  • 需要 0~1 之间输出的场景

**Sigmoid的作用：**提供非线性变换，使网络有表现复杂函数的能力

缺点：

  • 当x很大或者很小时，会导致梯度消失；
  • 输出范围0~1，均值不为0，会导致梯度更新方向不稳定

SiLU (Swish)

SiLU（Sigmoid Linear Unit），也叫 Swish，定义为： f(x)=x⋅σ(x)f(x) = x · σ(x)f(x)=x⋅σ(x)

  • 其中 σ(x)=1/(1+e−x)σ(x)=1/(1 + e^{-x})σ(x)=1/(1+e−x)

也就是说它是 输入 x 乘以 Sigmoid(x)

直观理解： f(x)=x⋅σ(x)f(x) = x · σ(x)f(x)=x⋅σ(x)

  • x 很大 → σ(x) ≈ 1 → 输出 ≈ x
  • x 很小 → σ(x) ≈ 0 → 输出 ≈ 0
  • x 接近 0 → 有一个平滑过渡

一般会和ReLU进行对比：max(0,x)max(0, x)max(0,x)

SiLU：

  • 不会突然截断
  • 是平滑的
  • 负数区域不是完全为 0，而是被“压小”

所以它是“平滑版 ReLU”。

为什么效果更好？

  • 关键在于平滑可导：ReLU 在 0 点不可导，而 SiLU 处处可导
  • 小负数不会完全丢失：ReLU 把负数全部砍掉；而 SiLU 允许小负值参与学习
  • 梯度更新稳定（无硬截断）

为什么 C2f 要做 Split？

C2f 是 YOLOv8 模型中的一个关键模块，主要用于增强特征提取能力，提高目标检测的精度和运行效率。它是一个残差块（Residual Block），通过引入残差连接（Residual Connection），优化了深度神经网络的训练过程。

C2f 模块的核心思想是让网络学习输入和输出之间的直接映射关系，从而缓解梯度消失问题，并提升网络的表示能力。其结构包含两个卷积层，中间通过残差连接相连。输入信号在残差连接中直接传递，同时通过卷积操作学习特征映射。

在C2f中：一部分特征值直接保留，一部分进入残差结构，最后再concat回来。

  • 参数：Kernel=5 × 5，Stride=1，Padding=2。串行执行 3 次。

SPPF 为什么要 连续执行 3 次？

简单来说，这样做可以获得不同尺度感受野，从而进行更好的特征融合，此外计算量也比传统的SPP小很多。

是什么？

作用：在一个局部窗口内取 最大值，是一种下采样/特征融合的操作

但如果学习的配置里：

这种设置 不会改变特征图大小。

输入输出形状：

输入：(N, C, H, W)

输出：(N, C, H_out, W_out)

计算公式：

  • Hout=H+2P−KS+1H_{out} = \frac{H + 2P - K}{S} + 1Hout​=SH+2P−K​+1

  • Wout=W+2P−KS+1W_{out} = \frac{W + 2P - K}{S} + 1Wout​=SW+2P−K​+1

其中：

  • K = kernel size
  • S = stride
  • P = padding

**核心点：**滑动窗口+局部最大值

SPPF 模块为什么要用 MaxPool？

在 SPPF 中：MaxPool(K=5,S=1,P=2)，会扩大感受野。

理解为：每个像素看到周围 5×5 区域信息

而 连续执行 3 次：5×5 → 9×9 → 13×13，感受野会逐渐扩大

在检测头里它的作用是：**把网络输出的一组 logits 转换为概率分布。**然后再用这个概率分布 计算期望得到最终的边界距离。

公式如下：

Softmax(xi)=exi∑jexjSoftmax(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}} Softmax(xi​)=∑j​exj​exi​​

为什么 DFL 需要 Softmax？

在 DFL（Distribution Focal Loss） 中：网络不会直接预测距离值，而是预测距离为i的概率，因此需要softmax计算的结果，进一步计算期望值，进而得出最终距离。

例如：针对预测结果：[logit0, logit1, logit2, logit3 … logit15]

Softmax 后得到：[0.01, 0.05, 0.20, 0.40, 0.25, …]

然后根据公式计算期望值：

0∗0.01+1∗0.05+2∗0.20+3∗0.40+4∗0.25=2.840*0.01 + 1*0.05 + 2*0.20 + 3*0.40 + 4*0.25= 2.840∗0.01+1∗0.05+2∗0.20+3∗0.40+4∗0.25=2.84

所以最终距离：distance = 2.84

这就是 DFL 的核心思想。
